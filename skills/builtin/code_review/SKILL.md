# 代码审查技能

## 任务
对代码进行全面审查，提供结构化反馈。

## 审查维度
1. 代码质量：命名规范、代码结构、可读性
2. 正确性：逻辑错误、边界条件、异常处理
3. 最佳实践：设计模式、SOLID原则、DRY原则
4. 性能：潜在性能问题和优化建议
5. 安全：常见安全漏洞检查
